# CLASS-ASSIGNMENT

단일 HTML로 동작하는 반편성 생성기입니다.

## 기능
- 학생 명렬표 줄바꿈 붙여넣기(엑셀 복사 → Ctrl+V)
- 동명이인 자동 구분(이름a/이름b/이름c ...)
- 반 수(K), 시드 기반 재현 가능한 랜덤
- 성별 입력(명렬표 표기 또는 남/여 목록 붙여넣기)
- 점수/등수(선택) 기반 반별 평균/분포 균등화
- 특성(특성별 이름 리스트 붙여넣기)
- 붙으면 안 되는 조합(이름1 - 이름2)
- 형제/쌍둥이 그룹(같은 반/다른 반/무시)
- 결과 복사 + CSV 다운로드
